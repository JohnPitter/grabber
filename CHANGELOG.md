# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.1.0] - 2026-02-08

### Added - Features

- Grabber frog mascot icon and favicon
- Local development server (`npm run dev:server`) - works without Firebase
- Video embed preview (YouTube iframe, Instagram embed) after fetching info
- `npm run dev:full` command to start frontend + backend together

### Changed - Branding

- Rebranded from "Media Grabber" to "Grabber" with frog theme

---

## [1.0.0] - 2026-02-08

### Added - Initial Release

- Initial project setup with Firebase + React + TypeScript
- Clean Architecture folder structure
- Firebase configuration (Functions v2, Hosting, Firestore, Storage)
- Vite + React 18 frontend with Tailwind CSS
- ESLint + Prettier code quality tooling
- Vitest test framework configuration
